Learning HTML And CSS

HTML

*What is HTML?*

             - HTML stands for Hyper Text Markup Language
             - HTML is the standard markup language for creating Web pages
             - HTML describes the structure of a Web page
             - HTML consists of a series of elements
             - HTML elements tell the browser how to display the content
             - HTML elements label pieces of content such as 
               "this is a heading", "this is a paragraph", "this is a link", etc.

*What is an HTML Element?*

               - An HTML element is defined by a start tag (< >), some content, and an end tag (< / >).    
               - The HTML element is everything from the start tag to the end tag

The content inside the body section will be displayed in a browser.
The content inside the title element will be shown in the browser's title bar or in the page's tab.

You should always include the lang attribute inside the  tag, to declare the language of the Web page.

*HTML Text Formatting*


                When you're writing bold text you can use : < b>< /b>
                When you're writing important text you can use : < strong>< /strong>
                When you're writing italic text you can use : < i>< /i>
                When you're writing emphasized text you can use : < em>< /em>
                When you're writing marked text you can use : < mark>< /mark> 
                When you're writing smaller text you can use : < small>< /small>
                When you're writing delted text you can use : < del>< /del>
                When you're writing inserted text you can use : < ins>< /ins>
                When you're writing subscript text you can use : < sub>< /sub>
                When you're writing supscript text you can use : < sup>< /sup>
            
                For example :          x2+y2=z2    or    log105


*HTML Tables*

                - Table Cells:
                        Each table cell is defined by a < td> and a < /td> tag.

                - Table Rows:
                        Each table row starts with a < tr> and end with a < /tr> tag.

                - Table Headers:
                        Sometimes you want your cells to be headers, in those cases use the < th> tag instead of the < td> tag.

CSS

*What is CSS?*

             - CSS is the language we use to style a Web page.
             - CSS stands for Cascading Style Sheets.
             - CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
             - CSS saves a lot of work. It can control the layout of multiple web pages all at once.
             - External stylesheets are stored in CSS files.
